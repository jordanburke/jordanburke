## Editing Rules

### Public-only content
This repo is public. **Only reference public OSS repositories** in the README. Never list, name, or link:
- Private repositories (from any org)
- Private org names
- Client or business-platform names that are not already publicly disclosed
- Internal filesystem paths or private knowledge-base locations

If in doubt, check whether a repo is visible at its GitHub URL while logged out. If not, it doesn't belong here.

### README structure
Group projects into clear categories (functional foundations, MCP servers, utilities). Prefer grouped tables over flat lists. Keep tone professional and concise — no emojis unless explicitly requested.

### When updating
1. Add new flagship-tier public repos promptly
2. Verify every link resolves before committing
3. Remove archived or dormant projects that no longer represent current work
4. Avoid precise counts that go stale (prefer qualitative phrasing)
